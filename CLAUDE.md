# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## リポジトリ概要

人生の方向性を明確にし、意思決定の一貫性を保つための自己分析リポジトリ。
価値観→現在地→方向性→判断基準→意思決定ログの順に、対話を通じて言語化していく。

## 構造

- `001_values/` — 核となる価値観（土台）
- `002_identity/` — 今の自分（現在地）
- `003_vision/` — どうなりたいか（方向性）
- `004_principles/` — 意思決定の原則（判断基準）
- `005_decisions/` — 意思決定ログ（実践・検証）
- `context/` — セッションごとのメモ

## 作業ルール

- 出力・ファイルへの記述内容は日本語で行うこと
- セッション終了時に `context/YYYY-MM-DD.md` にその日やったこと・次回やることを記録すること
- セッション開始時に `context/` の最新メモを読み、前回の続きから始めること
