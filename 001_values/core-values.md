# 核となる価値観

自分の人生の土台となる価値観。意思決定や行動の根底にあるもの。

## 1. 成長への誠実さ

安全圏に留まることより、課題に正面から向き合うことを選ぶ。
大学院を辞めて社会に出たのは、この価値観に基づいた最初の大きな決断だった。
意味のある困難は受け入れるが、意味のない苦痛には耐える必要はないと考える。

## 2. 自律と対等

誰かに可能性の天井を作られることを最も嫌う。
自分の意思で判断し、成果を出し、対等な立場で議論できることに強い価値を置く。
他者からの一方的な否定ではなく、互いにリスペクトのある関係を求める。

## 3. 人とのつながり

自由があれば友達と過ごすことを選ぶ。
孤独な環境には恐怖を感じる。
成功の姿にも常に「人との関係」が含まれている。

## 4. 影響力

自分の言葉が届き、人や社会を動かせる存在になりたい。
「何を言うか」だけでなく「誰が言うか」の力を手に入れたい。
最終的には、一人一人が自分に自信を持てる社会を作りたい。
