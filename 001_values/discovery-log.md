# 価値観の発見ログ

価値観を導き出すために行った問いと回答の記録。

## Q1. 心からやってよかったと思える経験

大学院を辞めて会社で働いていること。あのまま大学生を続けていたら今の自分の課題など一生向き合わないままだった。

→ **安全な場所より、課題に向き合える場所を選ぶ**

## Q2. 二度とやりたくない経験

水泳の合宿。単純にきつかった。水の中は孤独で怖かった。

→ **意味のない苦痛と孤独を嫌う。ただし意味のある困難は受け入れる**

## Q3. 他人に強く尊敬を感じるとき

自分の意思決定で大きな成果を上げている時。余裕が見える時。言葉に説得力がある時。

→ **自律的に成果を出し、説得力を持つ人に憧れる**

## Q4. 他人に強く嫌悪を感じるとき

天井を作られるような言動をされた時。自分の可能性を否定されるようなことを言われた時。

→ **他者の可能性を制限する行為を許せない**

## Q5. お金も時間も無限にあったら

友達と遊ぶ、日本を良くする。

→ **人とのつながり＋社会への貢献が根源的な欲求**

## Q6. 「日本を良くする」とは具体的にどんな状態か

一人一人が自分に自信を持っている状態。

→ **自分自身の課題（自信）と、解決したい社会課題が一致している**

## Q7. 発言に説得力がないと感じた場面

自分が言った言葉が正しくても否定された。他の人が同じことを言えば受け入れられるのに、自分が言うと馬鹿にされる。

→ **論理ではなく「誰が言うか」の力の不足を痛感している**

## Q8. 今の仕事を全力でこなしている理由

楽しいから。プレッシャーはある。言語化能力、コミュニケーション能力、仕事力の成長の場として使っている。

→ **仕事を成長の手段として主体的に活用している**

## Q9. 5年後の最高の状態

他人からリスペクトされている。自分に自信がある。仕事をバリバリやりつつ友達と遊んでいる。タワマンに住んでいる。

→ **実力・承認・人間関係・経済力が揃った状態**

## Q10. 言葉が届く人になったら最初に何をしたいか

自分の発言が受け入れられて相手と対等に話したい。相手の顔色を気にしないで議論したい。すごい人たちと対等に話し合えるようになりたい。

→ **影響力は支配のためではなく、対等な関係を築くために欲しい**
